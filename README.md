# ai-techniques-ml-neural-networks
Python project covering AI ethics analysis, logistic regression churn prediction, neural network flood risk forecasting, and NLP fake news classification using Keras and Scikit-learn.
## Overview

This project applies machine learning, deep learning, and AI ethics concepts 
across four real-world domains: recruitment bias, telecom churn, climate risk, 
and fake news detection.

---

## Question 1 — AI Ethics & Bias in Recruitment

Analysed a scenario where an AI hiring tool disproportionately rejected 
qualified female candidates.

- Identified three causes of gender bias in the model
- Outlined ethical and legal implications for the company
- Proposed technical and non-technical bias mitigation strategies

**Key concepts:** AI ethics, algorithmic bias, fairness-aware design

---

## Question 2 — Churn Prediction (Logistic Regression)

Built a logistic regression model to predict customer churn for a telecom 
provider, with a focus on precision due to the high cost of false positives.

- Loaded, preprocessed, and split the dataset
- Implemented logistic regression using Scikit-learn
- Evaluated model accuracy and precision with business justification
- Discussed limitations and recommended alternative models

**Key concepts:** Scikit-learn, logistic regression, precision/recall, 
feature scaling

---

## Question 3 — Flood Risk Forecasting (Neural Network)

Designed and implemented a feedforward neural network to classify flood 
risk in rural areas using climate-related features.

- Preprocessed data for neural network input
- Built and trained a Keras feedforward model (TensorFlow backend)
- Evaluated using precision and discussed its importance in disaster contexts
- Proposed strategies to improve model performance

**Key concepts:** Keras, TensorFlow, feedforward neural networks, 
binary classification

---

## Question 4 — Fake News Detection (NLP + Neural Network)

Built a neural network text classifier to distinguish fake news from real 
news based on article headlines.

- Preprocessed text using tokenisation and padding
- Built an embedding-based neural network using the Keras Sequential API
- Evaluated using accuracy and precision
- Discussed model limitations and recommended advanced architectures

**Key concepts:** NLP, text tokenisation, word embeddings, Keras, 
precision evaluation
